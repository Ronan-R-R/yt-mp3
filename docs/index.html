ytmp3

Pull audio straight off YouTube on your own machine. No upload sites, no ad walls, no server deciding what you can keep.

WindowsEXE / MP3 LinuxBIN / MP3 AndroidAPK / M4A
OUTPUT 192 kbps
Operation

Three moves and the file is yours.

01 / OPEN

Run the app

Download for your device and open it. On Windows, if it flags an unknown publisher, hit More info then Run anyway.

02 / PASTE

Drop a link

Copy a YouTube URL, paste it in the box, and choose where the file should land.

03 / RIP

Hit download

The audio pulls down and saves to your folder. Done. It plays anywhere, offline, forever.

Audio you have the rights to only. Respect the people who made it.
